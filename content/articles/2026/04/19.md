## `denoland/deno3-test1`

DenoのGitHub Organizationで以下の2つのリポジトリが公開されています。

- [denoland/deno3-test1](https://github.com/denoland/deno3-test1)
- [denoland/deno3-test2](https://github.com/denoland/deno3-test2)

Deno v3に関連したものと思われますが、まだ正式に機能が取り込まれるかどうかはわかりませんが、これらのリポジトリから読み取れる限りでは、[Deno.toml](https://github.com/denoland/deno3-test1/blob/3934d4678aa867550e90492b42d60e1fc0165bef/Deno.toml)というファイルでパッケージのメタ情報の定義や依存関係の管理などを行うアイデアが構想されているようです。

## Denoの直近の更新について

先週、Denoの _main_ ブランチにマージされている変更の中から主要なものについて紹介いたします。

### `deno upgrade pr <number>`

`deno upgrade`コマンドで特定のPR向けにビルドされたバージョンへのアップグレードがサポートされています。

- [feat(upgrade): support installing from a PR #33252](https://github.com/denoland/deno/pull/33252)

現時点では`gh run download`コマンドをベースに実装されているため、本機能の利用にはGitHub CLI (`gh`) がインストールされている必要あります。

```shell
$ deno upgrade --output ./deno-pr-33298 pr 33298

$ ./deno-pr-33298 --version
deno 2.7.12+4c90c4d (canary, release, x86_64-unknown-linux-gnu)
v8 14.7.173.19-rusty
typescript 5.9.2
```

### `deno audit` - Bulk Advisory Endpointへの移行

`deno audit`コマンドの内部実装において[Bulk Advisory Endpoint](https://github.com/npm/cli/blob/738be10651522e94038f1eff0da12f15b10438ea/docs/lib/content/commands/npm-audit.md#bulk-advisory-endpoint) (`/-/npm/v1/security/advisories/bulk`) へ移行する対応が行われています。

- [fix(audit): migrate to bulk advisory endpoint #33292](https://github.com/denoland/deno/pull/33292)

元々、[Full Audit エンドポイント](https://github.com/npm/npm-audit-report/blob/fbdb240694bd9a8d95e9386a157d471494a1426c/README.md#break-from-version-1) (`/-/npm/v1/security/audits`) というAPIが使われていましたが、直近のタイミングでこのBulk Advisory Endpointは廃止されたようで、その影響を受けての対応のようです。

本変更はDeno以外のパッケージマネージャーにも影響があったようで、[pnpmにおいても同様の対応が行われている](https://github.com/pnpm/pnpm/pull/11268)ようです。

## Optique v1

[Optique](https://github.com/dahlia/optique)というCLIパーサーライブラリの[v1.0.0](https://github.com/dahlia/optique/releases/tag/1.0.0)が公開されています。

- [Optique 1.0.0: environment variables, interactive prompts, and 1.0 API cleanup #796](https://github.com/dahlia/optique/discussions/796)
- [CHANGES.md - Version 1.0.0](https://github.com/dahlia/optique/blob/fad9b465d3382e5c43912b1fb5ac62907b84518e/CHANGES.md#version-100)

特徴の一つとして、npmに加えてjsrにもパッケージが公開 ([@optique](https://jsr.io/@optique)) されており、Node.jsだけでなくDenoなどのランタイムでも動作するようです。

本リリースでは新規パッケージとして[@optique/env](https://jsr.io/@optique/env) (環境変数のサポート用パッケージ) や [@optique/inquirer](https://jsr.io/@optique/inquirer) ([Inquirer.js](https://github.com/SBoudrias/Inquirer.js)との連携用ライブラリ) が追加されているようです。
