## Deno v2.6.8

[Deno v2.6.8](https://github.com/denoland/deno/releases/tag/v2.6.8)がリリースされています。

### Windows ARM64のサポート

DenoのWindows ARM64へのサポートが追加されています ([#31917](https://github.com/denoland/deno/pull/31917))

[GitHubのRelease](https://github.com/denoland/deno/releases/tag/v2.6.8)でもビルド済みのバイナリーが追加されています。

### jsr

[Deno v2.6.7]({{< ref "articles/2026/02/01.md" >}})における`@jsr`スコープの取り扱いの変更に続き、`package.json`の`dependencies`において`jsr:`スキームがサポートされています ([#31938](https://github.com/denoland/deno/pull/31938))

例えば、`package.json`で以下のように定義されている場合、[`jsr:@std/math@0.0.0`](https://jsr.io/@std/math@0.0.0)がインストールされます。

```json
{
  "dependencies": {
    "@std/math": "jsr:0.0.0"
  }
}
```

### `node:sqlite`

`DatabaseSync#createTagStore()` 及び`SQLTagStore`が実装されています ([#31945](https://github.com/denoland/deno/pull/31945))

また、`DatabaseSync`で`defensive`オプションのサポートが追加されています ([#32004](https://github.com/denoland/deno/pull/32004))

### `Worker`

`Worker`の各種コールバック (`onerror`, `onmessage`など) の型定義が誤っていた問題が修正されています ([#31981](https://github.com/denoland/deno/pull/31981))
